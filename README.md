- Get the project code

```bash
git clone https://github.com/lower-greenHand/vue3-PC-admin
```

- Installation dependencies

```
yarn 或 npm install 或 yarn install
```

- Run

```
yarn dev  或 npm run dev
```

- build

```
dev 环境
yarn build:dev

qa环境
yarn build:qa


pro
yarn build

```
